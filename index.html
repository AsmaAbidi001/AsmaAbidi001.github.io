Vocal Emotion Recognition System
Designed and implemented a complete DSP pipeline for automatic emotion classification from speech signals. Covers preprocessing (Butterworth filter, VAD), FFT/STFT/Mel spectrograms, 60-dimensional MFCC feature extraction, and KNN vs SVM classification. SVM with RBF kernel achieved 93.9% accuracy on RAVDESS. Results stored in SQLite with a 6-panel visualization dashboard. Also authored the full technical report for this project.
